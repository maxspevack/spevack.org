# spevack.org

Personal site for Max Spevack. Hosted on GitHub Pages.

## Stack

| File | Purpose |
|------|---------|
| `index.md` | Home |
| `resume.md` | Résumé (rendered at `/resume/`) |
| `_layouts/default.html` | Layout |
| `assets/css/vintage.css` | Theme — "Family Office" |

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

## Publishing

Push to `main`. GitHub Pages builds and deploys.

Print-to-PDF works from any browser via Cmd+P on `/resume/`.
